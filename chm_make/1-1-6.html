开发工具推荐

【Windows 下的开发工具推荐】：
AppServ: Apache MySQL PHP 整合安装包，安装后，通过命令 service.msc，可以看到 windows 服务会多出几项服务：apache 2.2, mysql
Beyond Compare：文件比较工具，非常清晰。
Zend Studio 5.5: Zend 官方出品，可以编辑和调试。
Chrome: 前端 DOM/JS 调试工具。
svn/git：版本控制工具。


【Linux 下的开发工具推荐】：
Zend Studio 5.5：跨平台哦。
vim：一般线上环境调试的时候，临时修改文件用。
EClipse：反应比较迟缓，不推荐使用。
Sublime：速度比较快，代码高亮提示都还可以，但不支持调试。
svn/git: 版本控制工具。
