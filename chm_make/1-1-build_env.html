一、安装 PHP 运行环境：

       1. WIN7 下：

              1.1 下载：APPSERV 2.5.9: http://x.big.xdowns.com/0710/appserv-win32-2.5.9.rar

              1.2 安装是选择路径：D:/appserv

              1.3 mysql 账号密码：root root

              1.4 安装后的配置文件的默认路径：

       php.ini：C:/windows/php.ini

       my.ini：D:/AppServ/MySQL/my.ini

       httpd.conf：D:/AppServ/Apache2.2/conf/httpd.conf

       WEB 主目录：D:/AppServ/www

       MySQL 数据目录：D:/AppServ/MySQL/data

 

                    

      

 2. Linux 下 ( CentOS )：

              2.1 顺序执行以下命令：

       cd /usr/local/src

       wget http://soft.vpser.net/lnmp/lnmp1.1-full.tar.gz

       tar zxvf lnmp1.1-full.tar.gz

       cd lnmp1.1-full

       ./centos.sh

 

              2.2 mysql 账号密码：root root

              2.3 安装后的默认路径：

       php.ini: /usr/local/php/etc/php.ini

       my.cnf：/etc/my.cnf

       nginx.conf: /usr/local/nginx/conf/nginx.conf

       WEB 主目录：/home/wwwroot

 

                    

                     MySQL 数据目录：/usr/local/mysql/var

                    

二、配置开发环境：

       1. 新建 Web 站点目录： x.com

              WIN7 下：

       mkdir D:/AppServ/www/x.com

 

              LINUX 下：

       mkdir /home/wwwroot/x.com

       chown -R www:www /home/wwwroot/x.com

 

       2. 增加一条 host 记录：127.0.0.1 x.com

              1.1 WIN7 下的方法：

                     1.1.1 管理员账户登录后，修改 c:/windows/system32/drivers/etc/hosts 的文件属性，去掉只读。

                     1.1.2 运行 notepad c:/windows/system32/drivers/etc/hosts

                     1.1.3 文件末尾增加一行：127.0.0.1 x.com

              1.2 LINUX 下的方法：

                     echo "127.0.0.1 x.com" >> /etc/hosts

       3. 增加 VirtualHost，同时增加 2 条 url-rewrite 规则

              2.1 Apache:

       <Directory D:/AppServ/www/x.com>

              Options FollowSymLinks ExecCGI Indexes

              AllowOverride None

              Order deny,allow

              Allow from all

              Satisfy all

       </Directory>

       <VirtualHost *:80>

              <IfModule mod_rewrite.c>

                     RewriteEngine on

                     RewriteRule ^(.*)/(.+?)\.htm$ $1/index.php?$1.htm [L]

              </IfModule>

              ServerAdmin x@x.com

              DocumentRoot "D:/AppServ/www/x.com"

              ServerName x.com

              ErrorLog "logs/x.com.log"

              CustomLog "logs/x.com-access.log" common

       </VirtualHost>

 

                    

 

              2.2 nginx:

       server {

              listen    80;

              server_name x.com;

              access_log    /home/wwwlogs/x_com.access.log combined;

              error_log     /home/wwwlogs/x_com.error.log crit;

             

              location / {

               index    index.html index.htm index.php;

               root    /home/wwwroot/x.com;

               location / {

                       rewrite "^(.*)/(.+?).htm$" $1/index.php?$2.htm last;

                       break;

               }

               if (!-e $request_filename) {

                       return 404;

               }           

        }

              location ~ \.php$ {

                  include fastcgi_params;

                  fastcgi_pass  unix:/tmp/php-cgi.sock;

                  fastcgi_index index.php;

                  fastcgi_param SCRIPT_FILENAME /home/wwwroot/x.com$fastcgi_script_name;

              }

              location /status {

                  stub_status on;

                  access_log   off;

              }

       }

 

                    

              2.3 重启 web server

                     2.3.1: apache:

                            运行：services.msc，找到 apache，点击“重启动此服务”

                     2.3.2: nginx:

       service nginx restart

      

              2.4 检查配置是否正确：

                     2.4.1: ping x.com，看是否域名已经被解析为 IP 127.0.0.1

                     2.4.2: 在 web 目录下新建文件 index.php，文件内容：<?php phpinfo(); ?>

                     2.4.3: 浏览器访问： http://x.com/ 看是否可以出现 PHPINFO 的页面

                     2.4.4: mysql -uroot -proot 看是否能登录 mysql
