$method

判断请求的方法，格式为：GET/POST/PUT/DELETE 。
判断浏览器发送的 HTTP 协议中的 METHOD 值。
【定义】 文件：xiunophp/xiunophp.php 大约 56 行： 
$method = $_SERVER['REQUEST_METHOD'];

【用例】 
<?php

include './xiunophp/xiunophp.php';

if($method == 'GET') {
	// GET 请求的时候，一般包含一个模板，输出 HTML
	include './page.htm';
} else {
	// 处理 POST 提交的数据
	print_r($_POST);
}

?>
