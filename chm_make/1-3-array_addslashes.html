array_addslashes(&$var)

addslashes() 的数组版本。
【定义】 文件：xiunophp/xiunophp.php 大约 264 行： 
function array_addslashes(&$var) {
	if(is_array($var)) {	
		foreach($var as $k=>&$v) {
			array_addslashes($v);
		}
	} else {
		$var = addslashes($var);
	}
	return $var;
}
