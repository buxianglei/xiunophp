array_assoc_slice($arrlist, $start, $length = 0)

从数组中取出一段，支持关联数组。
$arrlist: 二维数组
$start: 开始位置
$length: 取出长度
【定义】 文件：xiunophp/xiunophp.php 大约 359 行： 
function array_assoc_slice($arrlist, $start, $length = 0) {
	if(isset($arrlist[0])) return array_slice($arrlist, $start, $pagesize);
	$keys = array_keys($arrlist);
	$keys2 = array_slice($keys, $start, $length);
	$retlist = array();
	foreach ($keys2 as $key) {
		$retlist[$key] = $arrlist[$key];
	}

	return $retlist;
}
