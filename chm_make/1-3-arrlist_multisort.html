arrlist_multisort(&$arrlist, $col, $asc = TRUE)

对多维数组排序。

$arrlist: 二维数组
$col: Key
$asc: 排序方式
【定义】 文件：xiunophp/xiunophp.php 大约 319 行： 
function arrlist_multisort(&$arrlist, $col, $asc = TRUE) {
	$colarr = array();
	foreach($arrlist as $k=>$arr) {
		$colarr[$k] = $arr[$col];
	}
	$asc = $asc ? SORT_ASC : SORT_DESC;
	array_multisort($colarr, $asc, $arrlist);
	return $arrlist;
}
