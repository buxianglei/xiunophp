arrlist_values($arrlist, $key)

从一个二维数组中取出一个 values() 格式的一维数组，某一列 key 。

$arrlist: 二维数组
$key: Key
【定义】 文件：xiunophp/xiunophp.php 大约 394 行： 
function arrlist_values($arrlist, $key) {
	if(!$arrlist) return array();
	$return = array();
	foreach($arrlist as &$arr) {
		$return[] = $arr[$key];
	}
	return $return;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$arrlist = array(
	array('a'=>1, 'b'=>2),
	array('a'=>3, 'b'=>4),
);
$r = arrlist_values($arrlist, 'a');
print_r($r);

/*
Array
(
    [0] => 1
    [1] => 3
)
*/
?>
