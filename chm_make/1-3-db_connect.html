db_connect($err)

通过默认的 $db 实例连接数据库，成功返回 TRUE，失败返回 FALSE。
一般不需调用，XiunoPHP 的 DB 类在查询的时候回自动连接。
只有在需要测试是否可以连接成功才有必要。
一般只需 db_find(), db_find_one(), db_exec(), db_count() 直接使用就可以了。
【定义】 文件：xiunophp/xiunophp.php 大约 590 行： 
function db_connect(&$err) {
	global $db;
	$r = $db->connect();
	$err = $db->errstr;
	return $r;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

db_connect($err) OR exit($err);		// 测试看是否能连接成功

?>
