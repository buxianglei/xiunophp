db_count($table, $cond = array(), $abort = TRUE)

查询表的记录行数。

$table：表名
$cond：条件，默认为空是查询整个表的记录行数，速度最快
$abort：出错时是否终止流程
【定义】 文件：xiunophp/xiunophp.php 大约 544 行： 
function db_count($table, $cond = array(), $abort = TRUE) {
	global $db;
	$r = $db->count($table, $cond);
	if($r === FALSE && $db->errno != 0) {
		$s = "sql errno: ".$db->errno.", errstr: ".$db->errstr;
		xn_log($s, 'db_error');
		$abort AND xn_message(-1, $db->errstr);
	}
	return $r;
}
