encrypt($txt, $key = 'abcd9667676effff')

对字符串的可逆加密函数。根据 $key 随机生成，结果可以直接 URL 传递。
【定义】 文件：xiunophp/xiunophp.php 大约 881 行： 
function encrypt($txt, $key = 'abcd9667676effff') {
	return xn_urlencode(base64_encode(xxtea_encrypt($txt, $key)));
}
