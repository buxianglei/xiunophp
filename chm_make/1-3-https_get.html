https_get($url, $timeout=30, $cookie = '')

通过 HTTPS 方式请求 URL（默认 443 端口），成功返回页面内容，失败返回 FALSE。

$url: 请求 URL
$timeout: 超时秒数
$cookie: COOKIE
【定义】 文件：xiunophp/xiunophp.php 大约 1303 行： 
function https_get($url, $timeout=30, $cookie = '') {
	return https_post($url, $timeout, '', $cookie);
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$s = https_get('https://bbs.xiuno.com/');

?>
