humannumber($num)

将数值格式化为人们习惯阅读的格式。
【定义】 文件：xiunophp/xiunophp.php 大约 965 行： 
function humannumber($num) {
	$num > 100000 && $num = ceil($num / 10000).'万';
	return $num;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$r = humannumber(18518999);
print_r($r); // 1852万

?>
