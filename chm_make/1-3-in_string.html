in_string($s, $str)

判断一个字符串 $s 是否在另外一个字符串 $str 里面，分隔符为 , 。
【定义】 文件：xiunophp/xiunophp.php 大约 1436 行： 
function in_string($s, $str) {
	if(!$s || !$str) return FALSE;
	$s = ",$s,";
	$str = ",$str,";
	return strpos($str, $s) !== FALSE;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$r = in_string('2', '1,2,34667');	// TRUE

?>
