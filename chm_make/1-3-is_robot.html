is_robot()

根据 USER AGENT 判断是否为搜索引擎抓取内容的机器人。
【定义】 文件：xiunophp/xiunophp.php 大约 1171 行： 
function is_robot() {
	$browser = isset($_SERVER['HTTP_USER_AGENT']) ? strtolower($_SERVER['HTTP_USER_AGENT']) : '';
	$robots = array('bot', 'spider', 'slurp');
	foreach($robots as $robot) {
		if(strpos($browser, $robot) !== FALSE) {
			return TRUE;
		}
	}
	return FALSE;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$r = is_robot();	// TRUE|FALSE

?>
