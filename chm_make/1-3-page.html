page($page, $n, $pagesize)

获取页码数，防止超出。

$page: 当前页码数
$n: 最大记录行数
$pagesize: 每页记录行数
【定义】 文件：xiunophp/xiunophp.php 大约 935 行： 
function page($page, $n, $pagesize) {
	$total = ceil($n / $pagesize);
	$total < 1 AND $total = 1;
	return mid($page, 1, $total);
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$n = page(50, 201, 20);
