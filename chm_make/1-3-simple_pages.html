simple_pages($url, $totalnum, $page, $pagesize = 20)

简单的上一页，下一页，比较省资源，不用count(), 推荐使用。
【定义】 文件：xiunophp/xiunophp.php 大约 923 行： 
function simple_pages($url, $totalnum, $page, $pagesize = 20) {
	$totalpage = ceil($totalnum / $pagesize);
	if($totalpage < 2) return '';
	$page = min($totalpage, $page);

	$s = '';
	$page > 1 AND $s .= '上一页';
	$s .= " $page / $totalpage ";
	$totalnum >= $pagesize AND $page != $totalpage AND $s .= '下一页';
	return $s;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$s = simple_pages('user-list-{page}.htm', 100, 10, 5);
echo $s;

?>
