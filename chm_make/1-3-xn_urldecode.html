xn_urldecode($s)

解码 xn_urlencode() 编码的字符串。
【定义】 文件：xiunophp/xiunophp.php 大约 792 行： 
function xn_urldecode($s) {
	$s = str_replace('_', '%', $s);
	$s = urldecode($s);
	return $s;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';
