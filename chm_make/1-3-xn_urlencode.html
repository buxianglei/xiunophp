xn_urlencode($s)

对字符串进行编码 URL 字符串，结果可以用户 URL 传递，比较方便。
【定义】 文件：xiunophp/xiunophp.php 大约 782 行： 
function xn_urlencode($s) {
	$s = str_replace('-', '_2d', $s);
	$s = str_replace('.', '_2e', $s);
	$s = str_replace('+', '_2b', $s);
	$s = str_replace('=', '_3d', $s);
	$s = urlencode($s);
	$s = str_replace('%', '_', $s);
	return $s;
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';

$r = xn_urlencode('中国');
print_r($r); // _E4_B8_AD_E5_9B_BD

?>
