form_checkbox_yes_no($name, $checked = 0)

生成 checkbox 输入框 (yes/no) 。
【定义】 文件：xiunophp/form.func.php 大约 12 行： 
function form_checkbox_yes_no($name, $checked = 0) {
	$checked = intval($checked);
	return form_checkbox($name,  array(1=>'', 0=>''), $checked);
}

【用例】 
<?php

include './xiunophp/form.func.php';

$s = form_checkbox_yes_no('sex', '1');
echo $s;
// <label><input type="checkbox" name="sex" value="1" class="noborder" checked="checked" /></label>

?>
