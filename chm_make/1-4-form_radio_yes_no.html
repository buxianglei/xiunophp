form_radio_yes_no($name, $checked = 0)

生成选择按钮(是/否)。
【定义】 文件：xiunophp/form.func.php 大约 7 行： 
function form_radio_yes_no($name, $checked = 0) {
	$checked = intval($checked);
	return form_radio($name, array(1=>'是', 0=>'否'), $checked);
}

【用例】 
<?php

include './xiunophp/form.func.php';

$r = form_radio_yes_no('radio', 1);
print_r($r);

 */
