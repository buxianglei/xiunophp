form_textarea($name, $value, $width = 600, $height = 300)

生成多行的文本输入框。
【定义】 文件：xiunophp/form.func.php 大约 65 行： 
function form_textarea($name, $value, $width = 600,  $height = 300) {
	is_numeric($width) AND $width .= 'px';
	is_numeric($height) AND $height .= 'px';
	$s = "";
	return $s;
}

【用例】 
<?php

include './xiunophp/form.func.php';

$s = form_textarea('username', 1);
echo $s;
// <textarea name="username" id="username" style="width: 600px; height: 300px;">1</textarea>
