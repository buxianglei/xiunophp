image_get_dir($id)

根据 $id 取得图片路径。
【定义】 文件：xiunophp/image.func.php 大约 74 行： 
function image_get_dir($id) {
	$id = sprintf("%09d", $id);
	$s1 = substr($id, 0, 3);
	$s2 = substr($id, 3, 3);
	return "$s1/$s2";
}

【用例】 
<?php

include './xiunophp/image.func.php';

$s = image_get_dir(123456789);
echo $s; // 123/456/789

?>
