image_set_dir($id, $dir)

根据 ID 生成目录，最多 2 层，999/999，最多可以支持 1,000,000 个目录。

$id: 编号
$dir: 在哪个目录下生成
【定义】 文件：xiunophp/image.func.php 大约 62 行： 
function image_set_dir($id, $dir) {

	$id = sprintf("%09d", $id);
	$s1 = substr($id, 0, 3);
	$s2 = substr($id, 3, 3);
	$dir = $dir."$s1/$s2";
	!is_dir($dir) && mkdir($dir, 0777, TRUE);

	return "$s1/$s2";
}

【用例】 
<?php

include './xiunophp/image.func.php';

$r = image_set_dir(123, './upload/');

?>
