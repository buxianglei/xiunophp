$.fn.select_option(v)

这是一个 JQuery/Zepto 插件，它将值为 v 的 option 设置为选中状态。
【定义】 文件：xiunoui/xiuno.js 大约 1138 行： 
$.fn.select_option = function(v) {
	return this.each(function() {
		$(this).find('option').filter(function() {return this.value == v}).prop('selected', true);
	});
}

【用例】 <select id="select1"> <option value="1">value1</option> <option value="1">value2</option> <option value="1">value3</option> </select> <script> $('select1').select_option(1); </script>
