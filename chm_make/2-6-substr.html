substr(str, start, len)

截取字符串 str，start 为开始位置，len 为长度。
模拟 PHP 的 substr()，len 指定为负数的时候，从后开始计算长度。
【定义】 文件：xiunoui/xiuno.js 大约 275 行： 
global.substr = function(str, start, len) {
	// 支持负数
	var end = length;
	var length = str.length;
	if(start < 0) start = length + start;
	if(!len) {
		end = length;
	} else if(len > 0) {
		end = start + len;
	} else {
		end = length + len;
	}
	return str.substring(start, end);
}

【用例】 
var str = 'ABCDEF';
var s = substr(str, 1, 3);
console.log(s);
// BCD
