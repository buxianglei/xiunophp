plugin_install_replace($file, $old, $new)

将文件 $file 中的 $old 字符串替换为 $new 字符串。

$file: 要操作的文件
$old: 老的字符串
$new: 新的字符串
【定义】 文件：model/plugin.func.php 大约 160 行： 
function plugin_install_replace($file, $old, $new) {
	$s = file_get_contents($file);
	$s2 = str_replace($old, $new, $s);
	if($s != $s2) {
		return file_put_contents($file, $s2);
	} else {
		return xn_error(-1, '为找到指定的字符串，插件不能正常安装。');
	}
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';
include './model/plugin.func.php';

$file = './pc/view/header.inc.htm';
$old = '首页';
$new = '我的网站';

plugin_install_replace($file, $old, $new);

?>
