plugin_install_unshift($file, $s)

将 $s 插入到 $file 前端。

$file: 要操作的文件
$s: 在文件前端插入的代码
【定义】 文件：model/plugin.func.php 大约 103 行： 
function plugin_install_unshift($file, $s) {
	$old = file_get_contents($file);
	$new = $s.$old;
	if(strpos($old, $s) !== FALSE) return TRUE; // 如果已经存在，则不再追加
	return file_put_contents($file, $new);
}

【用例】 
<?php

$conf = include './conf.php';
include './xiunophp/xiunophp.php';
include './model/plugin.func.php';

$file = './pc/view/footer.inc.htm';
$code = '<script>alert(123);</script>';

plugin_install_unshift($file, $code);

?>
